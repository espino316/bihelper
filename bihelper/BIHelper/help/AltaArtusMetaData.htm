Nueva Conexión a Metadata de Artus

Funcionamiento

Registra en el sistema una conexión a una de base de datos MS SQL Server que contiene la información de Metadata de un servidor Artus.

Se capturan los siguientes datos:

Nombre El nombre de la conexión, como se identificará al servidor Artus en el sistema.
Servidor El nombre de la instancia de Sql Server que contiene la base de datos de Artus
Base de Datos Nombre de la base de datos que contiene la metadate de Artus
Usuario Nombre de usuario de MS Sql Server con permisos para consultar la base de datos que contiene la metadata de Artus
Password El password del usuario

El botón "Probar Conexión" sirve para probar la conectividad con el servidor antes de guardar la información en el sistema.

El botón "Guardar" registra la conexión del servidor Artus en el sistema.

El botón "Cancelar" cierra la ventana.
