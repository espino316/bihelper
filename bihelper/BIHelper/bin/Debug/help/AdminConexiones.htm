Administración de Conexiones

Funcionamiento

Administra las conexiones a servidores de metadata de Artus. Por el momento, solamente se soporta servidores MS SQL Server.

Las conexiones actuales aparecerán automáticamente en la tabla de conexiones.

Nueva Conexión

Para crear una nueva conexión, haz clic en "Capturar una nueva conexión".

Modificar una Conexión

Para modificar un conexión, haz clic en la columna "Actualizar" del renglón correspondiente a la conexión que deseas modificar.

Eliminar una Conexión

Para eliminar un conexión, haz clic en la columna "Eliminar" del renglón correspondiente a la conexión que deseas eliminar.
