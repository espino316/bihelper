Documentación de Cubo

Funcionamiento

Esta opción utiliza un conexión a un servidor de base de datos que contiene la metadata de Artus. Actualmente solo se soporta servidores de base de datos MS SQL Server como contenedores de metadata de Artus.

  1. Selecciona un servidor de metadata de Artus. Si no tienes uno configurado, da uno de alta.
  2. Al seleccionar o cambiar de servidor de metadate de Artus, apacecerán los cubos disponibles en la caja de lista desplegable "Seleccione un cubo".
  3. Selecciona el cubo que desee documentar.
  4. Haz clic en "Generar Reporte". La documentación del cubo se presetará en el visualizador de reportes.

*NOTA* Puedes exportar el reporte a Excel, haciendo clic en el ícono del diskette, y seleccionando la opción "Excel".
